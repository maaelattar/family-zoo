
# Family Zoo
 Family Zoo is a simple app that shows an interactive map of a zoo and habitats within it

## General info
Application has 6 habitats each habitat has a description and animals names within it, habitats names are:
  - AFRICA
  - BEAR DEN
  - BIG CATS
  - EAGLES NEST
  - GIANT BESTS
  - MONKEY HABITAT
